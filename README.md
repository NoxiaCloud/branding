# Brand Guidelines

### Website Theme

This theme primarily applies to our websites.

#### Colors

| Purpose                | HEX       | RGB           |
| ---------------------- | --------- | ------------- |
| **Background (Dark)**  | `#0f0f0f` | 15, 15, 15    |
| **Background (Light)** | `#f3f3f3` | 243, 243, 243 |
| **Accent**             | `#826cff` | 130, 108, 255 |

#### Attributes

- **Border / Corner Radius:** 8px
- **Font Family:** Space Grotesk
